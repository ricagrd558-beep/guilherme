# **App Name**: Fluxo Azul

## Core Features:

- Contabilização Rápida de Origem: Interface de um clique com botões grandes para o lojista registrar instantaneamente como o cliente chegou à loja (Redes Sociais, Passante, Recomendação ou Bairro).
- Dashboard Analítico Fintech: Visualização em tempo real das estatísticas de aquisição através de gráficos minimalistas e indicadores de performance.
- Persistência de Fluxo: Armazenamento seguro de cada registro de entrada no banco de dados para consulta de histórico e tendências.
- Consultoria de Marketing IA: Uma ferramenta que analisa os dados coletados e atua como um tool para sugerir em qual canal o lojista deve investir mais verba de marketing baseado no comportamento do cliente.
- Gerador de QR Code de Pesquisa: Cria um link único e QR Code para que o próprio cliente registre sua origem através do smartphone se preferir.

## Style Guidelines:

- Esquema de cores Dark Mode com fundo Azul Profundo (#0a0d14) para um visual sofisticado e tecnológico.
- Cor primária de ação: Azul Elétrico (#2264fa) utilizada em botões e destaques de navegação, inspirada no estilo fintech.
- Fonte 'Inter' para toda a interface, garantindo clareza absoluta e um aspecto moderno e limpo.
- Hierarquia visual baseada em cards com bordas arredondadas e sombras suaves, replicando a organização e minimalismo do app Nubank.
- Micro-interações fluidas e feedback visual imediato ao 'apertar' os botões de contabilização para confirmar o registro.
